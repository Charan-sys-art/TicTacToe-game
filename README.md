# TicTacToe Java Game 🎮

A simple console-based TicTacToe game built using Java.

## 👨‍💻 Features
- 2-player mode
- Win/tie logic
- Console UI

## 📁 How to Run
- Open project in Eclipse or any Java IDE
- Run the `TicTacToe.java` file

- Object-Oriented Programming
- Java basics
- Game logic & flow control
